Descrição: Aplicativo desktop feito para simular o recebimento de mensagens do módulo CubeSat AeroSpace, feito para o processo de trainee do grupo Cefast Aerospace
Uso: Para rodar o aplicativo sem a necessidade de compilação, basta executar o arquivo .jar no diretório "dir"
Integrantes: Isabela Costa e Samir

Observacoes:
Tarefa 1: Feita. Sempre que ha simulacao de leitura dos componentes uma simulacao da mensagem via sms e' exibida no console.

Tarefa 2: Feita. Classe BME, no pacote paginaSimulador. Nao e' preciso instancia-la pois os metodos para simular a medicao sao estaticos.

Tarefa 3: Feita. A imagem e' codificada na classe Imagem, no pacote paginaCodificarImagem. Nessa classe ha varias funcoes para se tratar uma 
imagem: codificar, decodificar, redimensionar, e comprimir uma string resultante de imagem codificada.

Tarefa 4: Parcialmente feita. Quando se codifica (e comprime) uma imagem, a string resultante pode ser enviada por SMS atraves do metodo quebrarString, classe SMS, pacote paginaSimulador. Entretanto, mesmo a string comprimida exige centenas (ou ate milhares) de mensagens SMS para enviar toda a informacao. Entao a chamada desta funcao foi comentada para evitar poluir o console.

Tarefa 5: Feita. E' necessario criar um login para utilizar o simulador.

Tarefa 6: Feita. A interface incrivel do simulador foi feita pela aluna e integrante Isabela Costa.
